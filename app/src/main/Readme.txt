Для полей TextView сверху и снизу отступ=FontSize/4,54 
                   слева и справа отступ=FontSize/10
Итого, чтобы было по 2 точки поля, нужно layout_marginTop = -FontSize/4,54 + 2
					 layout_marginBottom = -FontSize/4,54 + 2
                                         layout_marginLeft = -FontSize/10 + 2
					 layout_marginRight = -FontSize/10 + 2
2018.06.29 Версия 1.1. Добавлена проверка формата чисел. При удалении значений в полях в конфигурационной утилите показывался хинт. При этом значение было пустое и приложение вылетало с эксепшеном. Также сделана проверка на корректность цвета. Если цвет некорректный, то ставится значение по умолчанию. Это закрывает все падения приложения, которые были в версии 1.0 у пользователей.
